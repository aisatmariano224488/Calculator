# Simple Arithmetic Calculator

A clean, responsive web calculator built with **Vite**, **Vanilla JavaScript**, and **Tailwind CSS**.

## Features
* Perform basic arithmetic: Addition, Subtraction, Multiplication, and Division.
* Responsive design that works on mobile and desktop.
* Component-based architecture for modularity and maintainability.

## Getting Started

### Prerequisites
* [Node.js](https://nodejs.org/) (version 18+)
* npm or pnpm

### Installation
1. Clone the repository: `git clone <your-repo-url>`
2. Navigate to the folder: `cd calculator-app`
3. Install dependencies: `npm install`
4. Start the development server: `npm run dev`

## Built With
* [Vite](https://vitejs.dev/) - Build Tool
* [Tailwind CSS](https://tailwindcss.com/) - Styling
